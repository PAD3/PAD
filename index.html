get /students

Get all students

  • Response

Headers

  • Accept: required (string - default: application/json)

Query Parameters

  • q: (string)

    Query string

    Example:

    Jora
  • limit: (number)

    Limit for pagination

    Example:

    30
  • offset: (number)

    Offset for pagination

    Example:

    15
  • name_starts_with: (string)

    Search for students which name starts with this value

    Example:

    Jo
  • year_bigger_than: (number)

    Search for students who were born after this year

    Example:

    1995
